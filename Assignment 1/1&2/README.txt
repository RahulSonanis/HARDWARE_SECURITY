1) Extract the zip in the current folder.You will get a libsvm-3.21 folder. In that there is also a folder named matlab.
2) Open MATLAB. Open the libsvm-3.21->matlab folder in MATLAB.
3) compile using "make" command.
4) Just run the code using assgn command as "assgn.m" is the file name.
5) Now you will get two .txt files named as "first_part_exp.txt" and "second_part_sim.txt".These files contain the statistics of Machine Learning
   based modelling attack on APUFs.
